Սկզբից սկսեցի գուգլում գրած կոդերը նայել, ինչը մի քիչ բարդդ էր։
Հետո սկսեցի յութուբում հնդիկների վիդեոները նայել։ Ստեղ համեմատաբար հեշտ էր ամեն ինչ։ Էս տարբերկաը օգտագործելուց լիքը error ներ բերեց, որոնց չկարողացա գտնեմ թե ինչիցա ու տենց սկսեցի էլ հնդկական չնայել։
Հետո մի հատ անգլախոս տղա շատ պարզ բացատրեց արեցի, էլի error-ներ էղավ բայց 2-րդ օր ամեն ինչ ստացվեց։
